# 为什么制作？

# 到底为啥对咱不平凡

《时与涟》不仅仅是一款游戏，它是对我们母校的——无限怀念和敬意的致敬。它记录了我们从童年到青春的转变，那段留下了数不尽快乐和泪水的时光。

# 灵感来源

我们的故事始于最纯真的年代——幼儿园，那时母校的每一块石头，每一枚落叶，几乎在我们心中有了自己的生命。在学生时代简单而又快乐的日子里，我们享受着天真烂漫，学习看起来如此简单和愉快。走读生活中的朗朗读书声成了我们最美的乐章。

亲爱的母校，你见证了我们的成长，从天真的孩子蜕变为充满活力的青少年。初中时期虽然生活节奏加快，竞争增加，然而那份对学习与追求知识的渴望从未改变。它引导我们经历了无数的考验与挑战，使我们更坚韧，更成熟。

# 游戏的意义

制作《时与涟》对我们而言，不仅是对记忆的保存，也是一种挑战和探索。它检验了我们的耐心，考验了我们的创造力。在这个过程中，我们学会了如何协同工作，如何沟通和解决问题，为未来的人生旅途奠定扎实的基础。

游戏的开发既不是简单的业余爱好，也不是脱离现实的空想。它是一场对知识、技巧与合作的考验，是一个让每个参与者都得以专业提升的实战平台。

TIP

我们尽可能地让游戏设计与开发过程不干扰到日常学习，因此我们设立了宽松的工期，预计在明年暑假完成游戏的主体内容。

# 游戏内容

我们企图在《时与涟》中复现学校举办的每一次运动会，将七彩的风筝飞起，让操场上的欢声笑语响彻云霄；将每次画展里的每一幅作品，那些静谧中散发着温暖的色彩让人梦回校园；我们还将考试和人与人之间的矛盾与和解织入游戏的情节中，让玩家们不仅游历于校园的地理空间，更是在情感的海洋中遨游。

此外，为了提高游戏的趣味性和挑战性，我们融入了一些荒诞而合乎逻辑的元素。我们相信它会给玩家带来意想不到的惊喜和思考。

# 加入我们

如果你和我们一样，对母校有着说不完的故事和回忆，请不要犹豫。点击这里，访问参与贡献页面，与我们一起为《时与涟》添砖加瓦。无论是细微的建议，还是奇妙的新想法，我们都一样欢迎。

我们期待你的加入，让我们一起见证每一个版本的更新，每一个新想法的实现，因为《时与涟》记录的是我们大家的青春，我们共同的记忆。
